# Social Media Web Application

A **social media web application** inspired by platforms like **Instagram**, built using **HTML, CSS, and JavaScript**.  
This project focuses on front-end development concepts such as UI design, user interaction, and responsive layouts.

---

## 🚀 Features

- 👤 User profile interface  
- 🖼️ Photo & video post layout  
- ❤️ Like and 💬 comment UI  
- 📱 Responsive design for mobile & desktop  
- ⚡ Fast and lightweight (no backend)  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
